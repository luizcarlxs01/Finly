# Finly

Aplicativo de controle financeiro com modo sem conta e modo com conta.

## Estrutura

- `apps/web` → Next.js + shadcn/ui
- `apps/api` → .NET Web API
- `apps/mobile` → Flutter
- `docs` → documentação do projeto
- `.github/workflows` → workflows do GitHub Actions

## Branches principais

- `main`
- `luiz`
- `leo`

## Convenção de commits

- `feat:` nova funcionalidade
- `fix:` correção
- `refactor:` refatoração
- `docs:` documentação
- `chore:` tarefa técnica
- `style:` formatação
